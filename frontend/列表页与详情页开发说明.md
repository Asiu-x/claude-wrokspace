# 列表页与详情页开发说明

## 概述

已完成模型库、数据集、案例集、AI能力货架四个模块的列表页面和详情页面开发，包含筛选、搜索、分页等功能。

## 后端更新

### 1. 筛选功能增强
为四个模块的Controller添加了筛选和搜索参数支持：
- `ModelController`: 支持name、type、status筛选
- `DatasetController`: 支持name、category、status筛选
- `CaseController`: 支持title、category、status筛选
- `CapabilityController`: 支持name、category、status筛选

### 功能特性

#### 1. 列表页功能
- ✅ **搜索**: 支持按名称/标题搜索
- ✅ **筛选**: 支持按类型、状态等多维度筛选
- ✅ **分页**: 支持分页展示，每页条数可配置
- ✅ **重置**: 支持一键重置所有筛选条件
- ✅ **加载状态**: 数据加载时显示loading动画
- ✅ **空状态**: 无数据时显示友好提示
- ✅ **卡片布局**: 响应式网格布局展示

#### 2. 详情页功能
- ✅ **基本信息**: 展示实体的完整基本信息
- ✅ **状态显示**: 状态标签带颜色区分
- ✅ **性能指标**: 关键指标卡片展示
- ✅ **相关实体**: 展示关联的模型、数据集、能力等
- ✅ **标签/分类**: 标签和分类云展示
- ✅ **统计信息**: 访问量、下载量等统计
- ✅ **操作按钮**: 调用、下载、查看文档等操作
- ✅ **返回导航**: 一键返回列表页

### 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| / | Home | 首页 |
| /dashboard | Dashboard | 仪表盘 |
| /models | ModelsListPage | 模型库列表 |
| /models/:id | ModelDetailPage | 模型库详情 |
| /datasets | DatasetsListPage | 数据集列表 |
| /datasets/:id | DatasetDetailPage | 数据集详情 |
| /cases | CasesListPage | 案例集列表 |
| /cases/:id | CaseDetailPage | 案例集详情 |
| /capabilities | CapabilitiesListPage | AI能力列表 |
| /capabilities/:id | CapabilityDetailPage | AI能力详情 |

## 服务层

### 1. modelService.ts
```typescript
- getModels(params): 获取模型列表（分页、筛选）
- getModelDetail(id): 获取模型详情
```

### 2. datasetService.ts
```typescript
- getDatasets(params): 获取数据集列表（分页、筛选）
- getDatasetDetail(id): 获取数据集详情
```

### 3. caseService.ts
```typescript
- getCases(params): 获取案例列表（分页、筛选）
- getCaseDetail(id): 获取案例详情
```

### 4. capabilityService.ts
```typescript
- getCapabilities(params): 获取AI能力列表（分页、筛选）
- getCapabilityDetail(id): 获取AI能力详情
```

## 使用说明

### 启动项目

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器
npm run dev
```

### 访问页面

在浏览器中访问 http://localhost:5173，点击侧边栏导航即可进入不同模块的列表页和详情页。

## 依赖组件

列表页和详情页依赖以下公共组件：
- `FilterBar`: 搜索和筛选栏
- `Pagination`: 分页组件
- `EntityCard`: 实体卡片
- `EmptyState`: 空状态组件
- `MainLayout`: 主布局（包含Header和Sidebar）
