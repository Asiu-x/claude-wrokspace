# .claude 目录说明

本目录存放前端项目的开发规范、配置以及 Claude AI 辅助开发相关的文件。

## 文件说明
| 文件名 | 说明 |
|--------|------|
| `CLAUDE.md` | Claude AI 开发指南，包含项目基本信息、技术栈、目录结构等，供 Claude 了解项目使用 |
| `CODING_STANDARDS.md` | 代码规范，包含 TypeScript、React、样式、命名等开发规范 |
| `COMMIT_GUIDELINES.md` | Git 提交规范，约定式提交格式和要求 |
| `DEVELOPMENT_GUIDELINES.md` | 开发流程规范，包含环境准备、分支管理、开发步骤、API规范等 |
| `CODE_REVIEW_GUIDELINES.md` | 代码评审规范，包含评审标准、流程、职责等 |
| `memory/` | Claude 自动记忆目录，用于存储跨会话的记忆信息（自动生成，请勿手动修改） |

## 使用说明
1. 所有开发人员必须遵守本目录下的所有规范
2. 新增规范或修改现有规范需要经过团队评审
3. Claude AI 会自动参考这些规范进行代码生成和建议
4. 规范如有冲突，优先级：CLAUDE.md > 代码规范 > 提交规范 > 开发流程规范
