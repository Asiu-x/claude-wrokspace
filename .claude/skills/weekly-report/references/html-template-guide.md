# 飞书版 HTML 模板指南

## CSS 样式规范

所有HTML必须为单行压缩格式（标签之间不能有换行或空行），否则粘贴到飞书会出现多余空行。

### 核心样式

```css
body { font-family: -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif; max-width: 960px; margin: 0 auto; padding: 32px 24px; color: #1f2329; line-height: 1.5; background: #fff }
```

### 数据概览卡片

页面顶部显示4个数据卡片，分别统计：售前项目数、交付项目数、流程优化事项数、AI提效亮点数。使用 flexbox 布局，蓝色数字+灰色标签。

### 项目阶段标签配色

- **售前阶段**：`tag-blue`（蓝底蓝字 `#e8f0fe` / `#3370ff`）
- **交付/签约**：`tag-green`（绿底绿字 `#e8f8e8` / `#2ea121`）
- **变更/风险**：`tag-orange`（橙底橙字 `#fff3e0` / `#d4760a`）

### 风险项样式

- 无风险：`risk-none`（灰色 `#8f959e`）
- 有风险：`risk-warn`（红色加粗 `#d83931`）

### AI提效专项

使用 `highlight-box` 样式：蓝色渐变背景+蓝色边框圆角卡片，标题蓝色加粗。

## HTML 结构

```
报告头（姓名+周次+部门）
数据概览卡片栏
一、本周工作
  1. 售前项目表格
  2. 交付项目表格
  3. 流程支撑表格
二、AI提效成果专项（highlight-box）
三、下周计划
  1. 售前项目表格
  2. 交付项目表格
  3. 流程支撑表格
  4. AI提效计划表格
```

## 关键约束

1. 所有HTML标签必须压缩为单行，标签之间不允许有换行符
2. 报告头信息：`项绍栋 · 周报`，部门 `AI教学平台部`
3. 周次和日期范围根据实际日期动态计算
